University of York Bushcraft Society

About Us

Hello, we are the Bushcraft Society! We plan on running various activities on bushcraft, from knots to knife work to firelighting… eventually. COVID means that some activities which we would normally have to run in person can't be, but hopefully we’ll be able to run them before the end of this academic year!

In the meantime, we plan on running online events to teach some basic skills or to push existing skills further. Come along and join in or just have a chat in our Discord server
